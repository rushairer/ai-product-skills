# Codex Product Skills

Reusable Codex skills and templates for product-rule extraction and UI consistency review.

## Included Skills

- `app-rules-architect`
  - Extract recurring UX and product decisions into durable rules
  - Create and maintain files such as `APP_RULES.md` and `FEATURE_RULES.md`

- `product-ui-consistency-review-core`
  - Review sibling screens and repeated flows in a project-agnostic way
  - Focus on semantic, structural, interaction, and visual consistency

- `product-ui-consistency-review-specialized`
  - Review mature product families using domain-specific page patterns and terminology

## Included Templates

- `templates/APP_RULES_TEMPLATE.md`

## Included Docs

- `docs/SKILL_BOUNDARIES.md`
- `docs/RULES_AND_SKILLS_README.md`

## Structure

```text
skills/
  app-rules-architect/
  product-ui-consistency-review-core/
  product-ui-consistency-review-specialized/
templates/
docs/
```

## Installation

Copy any skill folder into your Codex skills directory:

```bash
~/.codex/skills/<skill-name>
```

Then restart Codex.
